FitLife API

Node.js · Express · Prisma · SQLite

Base URL

http://localhost:3000

Users

  • GET /users?limit=20&offset=0&search=alex – List users (pagination + search)
  • GET /users/:id – Get user details (includes workouts)
  • POST /users – Create user
  • PUT /users/:id – Update user
  • DELETE /users/:id – Delete user

POST /users body

{
  "firstName": "Alex",
  "lastName": "Lopez",
  "email": "alex@mail.com",
  "age": 22
}

Workouts

  • GET /workouts?limit=20&offset=0&search=cardio – List workouts (pagination + search)
  • GET /workouts/:id – Get workout details
  • POST /workouts – Create workout
  • PUT /workouts/:id – Update workout
  • DELETE /workouts/:id – Delete workout

POST /workouts body

{
  "title": "Cardio session",
  "description": "Treadmill and bike",
  "durationMinutes": 45,
  "userId": 1
}

Notes

  • Validation: required fields, numeric checks, firstName cannot contain digits.
  • Max limit is 100.
